# [漫长的相爱，是一场温柔的守望](https://shuiqiww.github.io/)

当青春的激情彻底褪去，当白发爬上鬓角，爱情会变成什么样子？

它变成了深夜为你掖好的被角，变成了病床前不离不弃的守候，变成了走过马路时下意识伸出的手。它不再需要玫瑰和烛光晚餐，它的仪式感融进了一粥一饭的日常里。

漫长的相爱，是一场温柔的守望。你们见证了彼此从青涩到成熟，从意气风发到坦然平和。你们熟悉对方如同熟悉自己掌心的纹路。你们依然是两个独立的个体，却在岁月的长河里，活成了彼此最深的羁绊和归宿。

这或许就是爱情最终极的浪漫：不是一时的狂热，而是一生的陪伴。当繁华落尽，我身边的人，依然是你。
